## Installation

```bash
# Example, adjust to the actual installation method
npm install -g logchange
# or
pip install logchange
```

## Quick Start

```bash
# Initialize logchange in your repository
logchange init

# Add your first change entry
logchange add "feat: initial feature"

# Generate changelog
logchange generate
```
